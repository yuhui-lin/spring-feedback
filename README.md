# spring-feedback
customer feedback application.

## Requirements

- Java 8
- maven 3
- git

## Running locally

```bash
git clone https://github.com/yuhui-lin/spring-feedback.git
cd spring-feedback
./mvnw spring-boot:run
# unit test
./mvnw test
```
You can then access the app here: http://localhost:8080/
